# ProtonVPN Connection Manager for Linux(Ubuntu)
A lightweight Python wrapper built on top of [`protonvpn_nm_lib`](https://github.com/ProtonVPN/linux-cli-community) that provides simple methods to **login**, **logout**, **connect**, **reconnect**, and **disconnect** from ProtonVPN servers.

Project URL: [View on GitHub](https://github.com/CasFre4/pyproton)
protonvpn_nm_lib 3.14.0: [Repo Download](https://repo.protonvpn.com/debian/dists/stable/main/binary-all/python3-protonvpn-nm-lib_3.14.0-1_all.deb)
proton_client 0.7.1: [Repo Download](https://repo.protonvpn.com/debian/dists/stable/main/binary-all/python3-proton-client_0.7.1-2_all.deb)
Copy of proton repo: [View on GitHub](https://github.com/CasFre4/ProtonPreserve)


---

## Features

- Login & Logout support  
- Connect 
- Reconnect
- Disconnect
- Built on top of `protonvpn_nm_lib`
